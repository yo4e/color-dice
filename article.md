# Color Dice — ランダムカラーパレット生成ツールを作りました

ウェブサイトやアプリのカラーテーマを考えるとき、「なんかいい感じの配色ないかな」って思うことありませんか？　私はよくあります。

そこで **Color Dice** を作りました。

👉 **https://colordice.yo4e.com**

---

## どんなツール？

サイコロを振る感覚で、ランダムにカラーパレットを生成できるツールです。

気に入った色があれば「ロック」して、他の色だけ再生成することもできます。生成したパレットは CSS / HTML / PNG 形式でエクスポートできるので、そのまま開発に使えます。

---

## 作った経緯

自分で使いたかったんです。

最初に相棒の月野テンプレクスがプロトタイプを作ってくれて、「これいいじゃん」となったので、Google Antigravity（AIコーディングアシスタント）に本格的に実装してもらいました。

4時間くらいで完成しました。AIすごい。

---

## 使い方

1. **ROLL**ボタンを押す（またはSpaceキー）
2. 気に入った色があれば🔒アイコンでロック
3. **Calm ⇄ Vivid** スライダで彩度を調整
4. **Dark ⇄ Light** スライダで明度を調整
5. **SAVE**でお気に入りに保存（最大20件）
6. **CSS / HTML / PNG** でエクスポート

スマホでは共有シートが開くので、そのままSNSに投稿したり、カメラロールに保存したりできます。

---

## 技術スタック

- **フロントエンド**: HTML / CSS / JavaScript（単一ファイル）
- **ホスティング**: Cloudflare Pages
- **共有機能**: Web Share API
- **データ保存**: localStorage

フレームワークなし、ビルドなし。シンプルに作りました。

---

## 💡 おまけ：画像生成AIとの連携

生成したCSSをNano Bananaなどの画像生成AIに渡して「このカラーパレットで画像を生成して」とお願いすると、統一感のあるビジュアルが作れます。めっちゃおすすめ！
